
This file documents changes made to the digitization of PD in which the
digitization is intentionally different than the printed edition.
The reasons for these changes are various.
This file was begun 04-04-2016, so there may be some changes prior to this
date which should be documented here but have not thus far been so 
documented.

-----------------------------------------------------------------------
04-04-2016
The reference  https://github.com/sanskrit-lexicon/CORRECTIONS/issues/108
contains details supporting these changes.
These changes initiated by Jim Funderburk.
In some cases, it was difficult to decide if a given change was due to an
error in the printing, or an error in the typist transcription of the printing.
